
# Sistema de Gestión de Datos en Python

## Descripción
Aplicación de consola desarrollada en Python que permite gestionar usuarios mediante operaciones CRUD,
validación de datos, uso de estructuras de control, funciones recursivas y modularización.

## Objetivos
- Implementar estructuras de control
- Aplicar funciones y recursividad
- Utilizar estructuras de datos (list, dict, tuple, set)
- Separar el código en módulos
- Exportar datos a CSV

## Arquitectura del sistema
gestion_datos/
├── main.py
├── funciones.py
├── validaciones.py
├── datos.py
└── usuarios.csv

## Tecnologías
- Python 3
- Biblioteca estándar (csv)

## Estructuras de datos utilizadas
- list: almacenamiento de usuarios
- dict: representación de cada usuario
- tuple: definición de campos inmutables
- set: control de IDs únicos

## Funcionalidades
- Agregar usuarios
- Listar usuarios
- Buscar usuarios
- Eliminar usuarios
- Contar registros (recursividad)
- Exportar CSV

## Ejecución
python main.py

## Buenas prácticas aplicadas
- PEP 8
- Modularización
- Validación de entradas
- Código reutilizable
